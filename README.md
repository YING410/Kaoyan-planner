# 考研通 MasterPlanner 

您好！这是一份由 AI 协助为您生成的考研打卡与日程规划软件源代码。

## 方案一：一键打包为独立 `.exe` 桌面软件（无需 Java 等任何环境）

我们已为您配置了基于 **Electron** 的专业打包方案。打包后的 `.exe` 程序自带了运行环境（类似自带了一个精简版浏览器核心），**任何电脑即使没有安装 Java 或 Node.js 也能直接双击运行！**

### 第一步：准备打包环境 (仅打包需要)
打包过程需要用到 Node.js，如果您的电脑尚未安装，请前往 [Node.js 中文官网](https://nodejs.org/zh-cn) 下载并安装 **LTS (长期支持版)**。

### 第二步：一键打包
1. 将在 AI Studio 导出的项目压缩包解压。
2. 双击运行文件夹内的 **`package-exe.bat`** 文件。
3. 等待几分钟，系统会自动构建并下载打包所需的资源。
4. 打包完成后，您可以在项目新增的 **`release`** 文件夹里找到生成的 `.exe` 安装程序！
5. 这个 `.exe` 您可以发给任何人，双击即可像普通 Windows 桌面软件一样安装、使用。

## 方案二：作为本地网页服务快速运行

如果您不想打包，只需自己用，也可以直接运行本地网页版。
1. 环境准备：安装 Node.js。
2. 运行：双击 **`run-win11.bat`** 文件，它会帮您安装依赖并在浏览器中自动打开应用。
